2.3 解构赋值

解构赋值是ES6中最灵活最有表现力的特性之一。同时，它也最为简单。它可以将对象的属性值绑定到任意多的变量上。解构赋值可以用于对象、数组以及函数参数列表中。

在ES5中，访问一个不存在的属性时，会返回undefined。

在解构赋值中，可以为这些值为undefined的属性解构提供默认值。默认值可以使任何类型的值：如数值、字符串、函数、对象或者对其他变量的引用等。

扩展运算符可以将可遍历对象转换为数组。扩展运算符能够方便地在数组或者函数调用中展开表达式。下例则在数组字面量中使用...arguments对函数的参数进行了转换。

使用扩展运算符可以将一个字符串分割成数组，数组中元素为组成字符串的每个字符。

扩展运算符的左右还可以添加其他内容，结果正如所期待的那样。

var all = [1, …[2, 3], 4, …[5], 6, 7] console.log(all) // <- [1, 2, 3, 4, 5, 6, 7]

值得一提的是，扩展运算符不仅能用于数组和arguments中，还可以用于任何可遍历对象中。可遍历是ES6中新引入的一种机制，它允许我们将对象转换成可被遍历的内容，我们将在第4章深入讨论这一部分的内容。
